# To-Do App

A simple and cute To-Do list iOS application built using Swift and Apple’s native tools.

The app allows users to manage daily tasks in a clean and minimal interface.

---

## Features

- Add tasks
- Delete tasks
- Mark/remove completed tasks
- Simple and intuitive UI
- Add lists
- Delete lists

---

## Built With

- Swift
- SwiftUI / UIKit (use whichever you actually used)
- Apple native development tools (Xcode)

---

## Assets

All images used in this project are sourced from Pinterest.

---

## How to Run

1. Install Xcode
2. Clone the repository
3. Open the project in Xcode
4. Run on the iOS simulator or a physical device
